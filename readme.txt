Task 1
При клике на кнопку меняется svg картинка внутри нее с помощью classList.toggle.

Task 2
При клике на кнопку вызывается alert, где при помощи window.screen.width и window.screen.height показывается ширина и высота экрана.

Task 3
Тут спрограммирован чат с использованием API WebSocket. С помощью функции writeMessage отправляется сообщение, написаное в инпуте, на сервер,
после чего приходит это сообщение с сервера, и все они отображаются в диалоговом окне. Вызов функции происходит при клике на кнопку "Отправить".
При клике на кнопку "Гео-локация" вызывается API geolocation, который при соглашении пользователя получает его координаты и выдает ссылку на его местоположение.
Иначе выводит сообщение о невозможности получить данные.
